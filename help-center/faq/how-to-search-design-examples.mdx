---
title: How do I search for design examples?
description: Find and filter design inspiration on Nicelydone.
---

## Search Features

Use the search bar at the top to enter keywords (e.g. "login", "dashboard").

## Filter by Category

- Click on any category: Flows, Screens, Patterns
- Use filters on the left (e.g. SaaS type, UI component, color)

## Sort Results

- Sort by Popularity or Latest using the dropdown above results.

## Example

Type `onboarding` in the search bar, then filter by `B2B SaaS` for relevant inspiration.
